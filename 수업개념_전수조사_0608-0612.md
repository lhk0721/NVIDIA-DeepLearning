### 등장 개념
- **시그모이드 함수** `y = 1 / (1 + np.exp(-x))` — 0~1로 압축, 이진분류 확률
- **SVM (서포트 벡터 머신)** 분류: `svm.SVC(C=, kernel='rbf', gamma=)`
  - `C` (cost) / `gamma` (감마) = 결정경계에 영향 주는 **하이퍼파라미터**
  - `kernel='rbf'` (방사 기저 함수 커널)
- **결정 트리** 분류: `tree.DecisionTreeClassifier(criterion='gini', max_depth, min_samples_leaf, min_samples_split, random_state)`
- **로지스틱 회귀** 분류: `LogisticRegression()` — 이름은 회귀지만 **분류** 모델
- **머신러닝 공통 흐름**: 데이터 준비 → 특성/타깃 분리 → train/test 분리 → (정규화) → 모델 생성 → `fit` → `score` → `predict`
- **표준점수 정규화(StandardScaler)**: (각 특성 − 평균) / 표준편차
- **원-핫 인코딩**: 범주를 0/1로만 표현
- **라벨 인코딩**: 문자열 라벨 → 정수

- `load_iris()` — 사이킷런 내장 붓꽃 데이터
  - 구조: `iris['data']`, `iris['target']`, `iris['target_names']`(setosa/versicolor/virginica), `iris['feature_names']`
- `pd.DataFrame(np.column_stack([...]), columns=[...])` — 배열 합쳐 데이터프레임 만들기

## 📅 6/9 — 다중분류 + softmax (딥러닝 진입 직전)

### 파일 보는 순서
1. `softmax_exam.py` — 소프트맥스 함수 그 자체
2. `fish_data_다중분류.py` — 물고기 7종 다중분류 (로지스틱 회귀)

### 등장 개념
- **softmax 함수** = 정규화된 지수함수
  - 여러 출력값을 0~1로 압축 + **전체 합 = 1** (확률 분포)
  - 식: `exp(z) / sum(exp(z))`
- **다중분류(multi-class)**: 3개 이상 클래스 분류 (여기선 물고기 7종)
- **multinomial 로지스틱 회귀**: 로지스틱 회귀로 다중분류 하기
- `predict_proba` — 클래스별 확률 출력 (vs `predict` 는 클래스만)

### 함수 / 메서드 / 스킬
- `np.exp(arr)`, `np.sum(exp_a)` — 지수/합
- `np.argmax(y)` — **가장 큰 값의 인덱스** 반환 (확률 최대 클래스 찾기)
- `classdata[np.argmax(y)]` — 인덱스로 클래스 이름 색인
- `np.round(y, decimals=3)` — 반올림
- `np.set_printoptions(precision=8, suppress=True)` — 지수표기 대신 소수점 출력
- `df['col'].unique()` — 고유값(클래스 종류) 확인
- `df['col'].to_numpy()` — 시리즈 → 넘파이 배열
- `df.columns` — 컬럼명 확인
- `LogisticRegression(multi_class='multinomial', max_iter=1000, C=20)` ★ 다중분류 설정
  - `max_iter` = 반복 횟수 / `C` = 규제 강도
- StandardScaler 정규화 (fit_transform / transform)
- `model.score()`, `model.predict()`, `model.predict_proba()`, `model.classes_`
- 마지막 줄 `from tensorflow.keras.models import Sequential` → **딥러닝(케라스)으로 넘어가는 신호**

## 📅 6/10 — 딥러닝 모델 설계 기초 (케라스 Sequential · 선형회귀)

### 파일 보는 순서
1. `딥러닝모델_설계_예제1.py` — Sequential / Dense 최소 골격 + summary
2. `딥러닝_선형회귀_예제.py` — 선형회귀 모델 설계 + 학습/가중치 확인(주석 포함)
3. `딥러닝_선형회귀_예제2.py` — 다층(은닉층) 신경망 + compile/fit/evaluate
4. `딥러닝_보스턴주택가격_선형회귀.py` — **빈 파일(내용 없음)**, 제목만 (보스턴 주택가격 회귀 예고)

### 등장 개념
- **Sequential 모델** = 층을 순서대로 쌓는 딥러닝 전체 틀
- **Dense (완전연결층)**: `Dense(units, input_dim, activation, use_bias)`
  - `units` = 그 층의 **뉴런 개수**
  - `input_dim` = 입력 데이터 개수(첫 층에만 지정)
  - **첫 층 = 입력층**, 입력 형태를 반드시 알려줘야 함
- **활성화 함수(activation)** 종류
  - `'linear'` — 회귀 출력층
  - `'sigmoid'` — 이진분류 출력
  - `'leaky_relu'`, `'relu'` — 은닉층
- **딥러닝 학습 3단계**: ① 모델 쌓기(add) ② `compile` ③ `fit`
- **손실함수(loss)**: 회귀는 `'mse'` (평균제곱오차)
- **옵티마이저(optimizer)**: `'adam'`
- **평가지표(metrics)**: `'mae'` (평균절대오차)
- **에포크(epochs)** / **배치 사이즈(batch_size)** 개념
- **다항 특성 추가**: 길이² 특성 추가해 비선형 표현 (`x**2` 컬럼 추가)

### 함수 / 메서드 / 스킬
- `model = Sequential()` → `model.add(Dense(...))`
- `model.summary()` — 모델 구조/파라미터 수 확인
- `model.compile(loss='mse', optimizer='adam', metrics=['mae'])`
- `model.fit(X, Y, batch_size=1, epochs=1000, verbose=1)`
- `model.evaluate(test_x, test_y)` — 손실/지표 평가
- `model.predict(...)` — 예측
- **가중치 직접 확인 스킬**: `model.layers[0].get_weights()` → `w = weights[0][0][0]`
- `use_bias=False` — 절편 사용 안 함 옵션
- `Dense(units=1, input_dim=1, activation='linear')` (입력층 지정 방법 1)
- `Input(shape=(1,))` 로 입력층 따로 추가하는 방법(주석)
- 넘파이 데이터 만들기 스킬
  - `np.linspace(0,10,10)`, `np.random.randn(*X.shape)` (노이즈 추가)
  - `arr.reshape(-1,1)` — 1차원→2차원(열벡터)
  - `np.column_stack((train_x**2, train_x))` — 다항 특성 합치기
- 회귀선 시각화(주석): `plt.plot(X, Y, label='data')`, `plt.plot(X, w*X, label='pred')`

5. `딥러닝_iris_붓꽃_다중분류.py` — **데이터 로드만(미완성)**, 붓꽃 다중분류 예고

### 등장 개념
- **다중분류 딥러닝 출력층**: `Dense(클래스수, activation='softmax')` + loss `'categorical_crossentropy'`
- **이진분류 딥러닝 출력층**: `Dense(1, activation='sigmoid')` + loss `'binary_crossentropy'`
- **정답 원핫 인코딩 필요**: categorical_crossentropy 쓰려면 정답이 원핫이어야 함
- **검증 데이터(validation_data)** 로 과적합 모니터링
- **콜백(callback)** 개념 ★
  - **ModelCheckpoint**: val_loss 개선될 때만 모델 저장
  - **EarlyStopping**: 개선 없으면 조기 종료
- **모델 저장/불러오기** (`.keras` 포맷)
- **스케일러 저장/불러오기** (`joblib`) — 예측 때 학습과 동일한 정규화 재사용

### 함수 / 메서드 / 스킬
- `LabelEncoder().fit_transform()` → `to_categorical(y_encoded)` (원핫 변환)
  - `from tensorflow.keras.utils import to_categorical`
- 모델 구조
  - 다중: `Dense(10, input_dim=5, activation='leaky_relu')` → `Dense(7, activation='softmax')`
  - 이진: `Dense(8, input_dim=4, 'leaky_relu')` → `Dense(4,'leaky_relu')` → `Dense(1,'sigmoid')`
- `compile(loss='categorical_crossentropy'/'binary_crossentropy', optimizer='adam', metrics=['accuracy'])`
- **콜백 코드 스킬**
  - `ModelCheckpoint(filepath='./fish_bestmodel.keras', monitor='val_loss', verbose=1, save_best_only=True)`
  - `EarlyStopping(monitor='val_loss', patience=3, restore_best_weights=True)`
  - `model.fit(..., validation_data=(val_x,val_y), callbacks=[checkpoint_cb, earlystop_cb])`
- `model.evaluate(test, test_y)[1]` — `[1]`로 정확도만 뽑기
- `model.save('이름.keras')` / `load_model('이름.keras')`
- **joblib 스킬**: `joblib.dump(scaler, 'fish_scaler.pkl')` / `joblib.load('fish_scaler.pkl')`
- **예측 스킬**: 새 데이터 → `scaler.transform()` → `model.predict()` → `np.argmax(pred, axis=1)` → 클래스배열로 색인
- `np.set_printoptions(threshold=np.inf)` — 배열 생략 없이 전부 출력
- 타이타닉 전처리는 6/8과 동일(map/dropna/get_dummies/concat/StandardScaler)
- ⚠️ 작은 함정 팁: 예측 파일이 `fish_multi_clf.keras`(주석 처리됨)를 불러오는데 학습 파일은 `fish_bestmodel.keras`로 저장 → **저장 파일명/주석 일치 확인 필요**

## 📅 6/12 — CNN 이미지 분류 (fashion_mnist)

### 파일 보는 순서
1. `fashin_mnist_딥러닝_분류.py` — **CNN 모델 설계·학습** (핵심)
2. `fashion_minst_딥러닝_예측.py` — 테스트셋 일부 예측
3. `fashion_mnist_임의데이터예측.py` — **실제 외부 이미지(sandal_1.jpg)** 예측 (opencv 전처리)

### 등장 개념
- **CNN (합성곱 신경망)** — 이미지 분류 표준 구조
- **Conv2D (합성곱층)**: `Conv2D(filters, kernel_size, activation, padding, input_shape)`
  - `filters` = 필터 개수 / `kernel_size=3` → 3×3
  - `padding='same'` = zero padding(출력 크기 유지)
  - 첫 층은 `input_shape=(28,28,1)` 지정
- **MaxPooling2D(2)** — 2×2 영역 최대값 (다운샘플링)
- **Flatten** — 2차원 특성맵 → 1차원 (FC층 연결용)
- **Dropout(0.4)** — 과적합 방지(뉴런 일부 끔)
- **이미지 정규화**: `/255.0` (0~1로)
- **이미지 채널 차원**: `reshape(-1, 28, 28, 1)` (흑백 채널 1)
- **출력층**: 클래스 수(10)만큼 뉴런 + `softmax`
- **fashion_mnist**: 10종 의류, train 60000 / test 10000

### 함수 / 메서드 / 스킬
- `from tensorflow.keras.datasets import fashion_mnist`
- `(train_x, train_y), (test_x, test_y) = fashion_mnist.load_data()`
- `np.unique(train_y, return_counts=True)` — 클래스별 개수 확인
- 데이터 일부만 언패킹: `(_, _), (test_x, _) = fashion_mnist.load_data()` (밑줄로 버리기)
- `train_test_split(..., test_size=0.2)` 로 train/validation 분리
- `to_categorical(train_y)` — 원핫
- **CNN 모델 쌓는 순서** (전형적 패턴 암기 포인트)
  - Conv2D → MaxPooling2D → Conv2D → MaxPooling2D → Flatten → Dense → Dropout → Dense → Dense(softmax)
- `model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])`
- 콜백(ModelCheckpoint/EarlyStopping)으로 학습 (6/11과 동일 패턴)
- `history = model.fit(...)` — 학습 기록 반환
- 이미지 띄우기 팁(주석): `plt.imshow(train_x[0], cmap='gray')` (0=검정, 255=흰색)
- **예측 스킬**: `pred = model.predict(test_scaled[:5])` → `np.array(classes)[np.argmax(pred, axis=1)]`
  - 클래스: `['티셔츠','바지','스웨터','드레스','코트','샌달','셔츠','스니커즈','가방','앵클부츠']`
- **외부 이미지 예측 전처리(OpenCV) 스킬** ★ (가장 실전적)
  - `cv2.imread(경로, cv2.IMREAD_GRAYSCALE)` — 흑백 로드
  - `cv2.resize(img, dsize=(28,28), interpolation=cv2.INTER_AREA)` — 모델 입력 크기로 리사이즈
  - `cv2.bitwise_not(img)` — **색 반전** (mnist는 배경 검정/사물 흰색이라 반전 필요)
  - `/255.0` 정규화 → `reshape(1,28,28,1)` → `model.predict()`
  - `np.set_printoptions(precision=3)` 등 출력 옵션
